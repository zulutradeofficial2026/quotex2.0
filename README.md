# Quotex Pro 2.0 -  Trading app

## 
**This is a trading simulation for real tradinglike quotex website  purposes only.**
- USdt real money is involved.
- Charts and price movements are  simulated .
- "AI Signals" are for demonstration of platform mechanics only.

## Overview
Quotex Pro 2.0 is a next-generation trading simulator designed to provide a "Ghost Mode" trading experience. 

### Features
- **Stunning UI**: Dark "Ghost Mode" theme with neon accents.
- **Custom Chart Engine**: 60fps canvas-based candlestick chart with smooth animations.
- **Offline Simulation**: Works completely offline without external API dependencies.
- **AI Integration**: Demonstrates AI signal analysis concepts.
- **Timezone Logic**: Features localized features for Pakistan Standard Time (PKT).

## How to Run
1. Open `index.html` in any modern web browser.
2. No server required (it's a static single-page application).
